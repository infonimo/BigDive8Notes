#### | pipe
use the output of the command on the left as the input to the command on the right.

```
ps aux | grep python
```
#### > redirect
redirects a command’s output to a file (overwriting any existing content).

\>> appends a command’s output to a file.
#### Wildcards
+ \* any number of char
+ [AB] either A or B
